# Plugins

Los complementos abarcan desde la optimización de paquetes y la minificación hasta la definición de variables similares al entorno.

**Loaders** : intérpretes, parsers y pre-procesadores

**Plugins** : librerías, métodos que aportan funcionalidades particulares en JS al entorno de webpack

más info [aqui](https://webpack.js.org/concepts/#plugins)
